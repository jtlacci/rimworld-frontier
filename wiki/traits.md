# Traits

Colonists have 1-3 traits that permanently affect their stats, mood, and behavior. Traits cannot normally be gained or removed after character creation.

## Work Speed Traits (Highest Impact)

| Trait | Work Speed | Other Effects |
|-------|-----------|---------------|
| Industrious | **+35%** | Best general productivity trait |
| Hard Worker | +20% | |
| Very Neurotic | **+40%** | +14% mental break threshold (risky) |
| Neurotic | +20% | +8% break threshold |
| Lazy | -20% | |
| Slothful | **-35%** | Massive productivity loss |

Best combo: Very Neurotic + Industrious = +75% work speed (but high break risk).

Base: Minor 25%, Major 15%, Extreme 5%.

| Trait | Threshold Change |
|-------|-----------------|

| Steadfast | -6% |
| Nervous | +8% |
| Volatile | **+10%** (very fragile) |

## Agent Decision Guide

### Green Flags
- **Industrious**: +35% work speed (best general trait)
- **Tough**: x0.50 damage (best combat trait)
- **Iron-willed**: -12% break threshold (most stable)
- **Sanguine**: +12 mood (permanent buffer)
- **Fast Learner**: +75% learning (rapid skill growth)
- **Jogger**: +0.4 movement (faster everything)

### Red Flags
- **Pyromaniac**: starts fires, can't fight fires
- **Chemical Fascination**: drug binges
- **Volatile**: +10% break threshold
- **Slothful**: -35% work speed
- **Slow Learner**: -75% XP (won't improve)
