# Backstories and Incapabilities

Every colonist has a childhood backstory, and adults (20+) also have an adulthood backstory. Backstories grant skill bonuses, passions, and work type restrictions.

## Work Type Incapabilities

| Incapability | Disables |
|-------------|---------|
| ManualDumb | Hauling, cleaning, plant cutting |
| ManualSkilled | Construction, cooking, crafting, smithing, tailoring, art |
| Violent | ALL combat (shooting, melee), hunting |
| Caring | Doctoring, wardening |
| Social | Wardening, trading, recruitment |
| Intellectual | Research |
| Crafting | Smithing, tailoring, crafting |
| Cooking | Cooking |
| Cleaning | Cleaning |
| Hauling | Hauling |
| PlantWork | Growing, plant cutting |
| Artistic | Art creation |
| Mining | Mining |

## Problematic Patterns

- **Noble/Royal backstories**: often incapable of ManualDumb (no hauling/cleaning)
- **Artist backstories**: often incapable of ManualDumb + Crafting + Cooking
- **Intellectual backstories**: often incapable of ManualDumb
- **Some combat backstories**: may be incapable of Caring or Social

## Backstory Categories

- **ImperialCommon**: Servants, priests, entertainers (social/artistic focus)
- **ImperialFighter**: Soldiers, engineers, medics (combat + support)
- **ImperialRoyal**: Leaders, nobles (high skills, many incapabilities)
- **Glitterworld**: Scientists, surgeons, officers (high intellectual/medical)
- **Medieval**: Farmers, craftsmen, sailors (manual labor focus)
- **Outsider**: Mixed civilian (most versatile, fewest restrictions)

## Agent Evaluation Rules

1. **Check incapabilities first** — a pawn incapable of Violence cannot fight, period
2. **Passion > current skill** — level 3 with burning passion outpaces level 8 with none within a season
3. **Traits dominate** — Industrious (+35% work) is worth more than 2-3 extra skill levels
4. **Versatility matters** — a pawn with no incapabilities can fill any gap
